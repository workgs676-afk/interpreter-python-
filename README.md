# Web Python Interpreter (HTML-Based)

## Overview
This project is a simple web-based Python interpreter built using HTML, CSS, and JavaScript. It enables users to run Python code directly in the browser without installing Python locally, making it ideal for learning, practice, and quick experimentation.

## Features
- Run Python code directly in the browser
- Instant output display
- Simple and beginner-friendly interface
- No installation required
- Great for learning and testing small scripts

## How It Works
The interpreter uses a browser-based Python runtime (such as Pyodide or similar) to execute Python code. The interface is built using:

- HTML → Structure of the editor and output area
- CSS → Styling and layout
- JavaScript → Handles code execution and interaction

## Usage
1. Open index.html in your browser
2. Write Python code in the editor
3. Click the "Run" button
4. View the output below

## Example
print("Hello, World!")

Output:
Hello, World!
